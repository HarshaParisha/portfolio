# Portfolio Images

This directory contains images for the portfolio website.

## Required Images:

- `profile.jpg` - Your profile photo (320x320px recommended)
- `og-image.jpg` - Open Graph image for social media sharing (1200x630px)
- `ecommerce-project.jpg` - Screenshot of e-commerce project
- `task-app-project.jpg` - Screenshot of task management app
- `weather-project.jpg` - Screenshot of weather dashboard

## Placeholder Images:

For now, placeholder images are used. Replace these with actual project screenshots and your professional photo.

## Image Optimization:

- Use Next.js Image component for automatic optimization
- WebP format recommended for better performance
- Include alt text for accessibility
